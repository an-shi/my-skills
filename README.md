# my-skills

我的自制 AI Skill 合集。

## Skills

### task-doc-writer

为当前任务编写 Markdown 文档，并保存到项目根目录的 `.docs` 文件夹中。

适合场景：

- 总结一次编码、维护、调试或配置任务
- 记录本次任务涉及了哪些文件
- 说明每个文件为什么被创建、修改、移动或删除
- 将任务验证情况沉淀到项目文档中

文档命名格式：`YYMMDD-标题.md`，例如 `260601-document-1.md`。

路径约定：文档中的项目文件路径从工作区或项目根目录开始，不写本机绝对路径。

## 安装

将需要的 skill 目录复制到对应工具的全局 skills 目录：

- Codex：`C:\Users\bw51\.codex\skills`
- Trae：`C:\Users\bw51\.trae-cn\skills`
